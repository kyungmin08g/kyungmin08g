### ✨ About

코드는 나의 표현 방식이고,  
서비스는 내가 세상을 이해하는 방법입니다.

중학교 2학년 때 처음 개발을 시작했고,  
지금은 단순한 구현을 넘어 **구조와 확장성**을 고민하는 단계에 있습니다.

Java, Kotlin, Spring을 중심으로  
“잘 동작하는 코드”가 아니라  
“오래 살아남는 시스템”을 만들고 싶습니다.
